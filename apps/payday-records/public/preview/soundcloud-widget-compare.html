SoundCloud widget — UI compare

payday-records 플레이리스트 섹션에서 어떤 UI를 쓸지 결정용. URL은 juntaromusic / sets / no-time-to-talk 동일.

이 파일은 비교용 스크래치이며 PR 머지 전 삭제 예정입니다. 접속: pnpm dev --filter=payday-records 후 http://localhost:3002/preview/soundcloud-widget-compare.html

1. Visual mode (현재 선택)

visual=true · height=450 — 큰 아트워크 배경 + 트랙 리스트 오버레이

2. Classic mode (셋, 트랙 리스트 펼침)

visual=false · height=300 — SoundCloud 시그니처 파형(waveform) + 좌측 아트워크

3. Mini mode

visual=false · height=20 — 진행 막대 + 재생 버튼만

4. Visual + 사이트 톤 색상 (#A07CFE 보라)

visual=true · color=A07CFE — color 파라미터의 영향만 비교

5. Classic + 색상 (#A07CFE)

visual=false · height=300 · color=A07CFE
